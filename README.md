# Neuro Nav PC (Hackathon Build)

Desktop-first web app inspired by the Neuro Nav concept for autism-friendly navigation and sensory support.

## Run locally

1. Open `index.html` directly in a browser, or serve with any static server.
2. Copy `firebase-config.example.js` to `firebase-config.js` and fill project values.
3. If `firebase-config.js` is missing, the app runs in local storage mode.
4. For Google sign-in, enable Google provider in Firebase Authentication.

## Feature roadmap in this repo

- Panic/meltdown support button with geolocation + caregiver SMS payload
- Sensory-friendly route planning + safe havens with Google Maps links
- Environmental analysis and calm score monitoring
- Music therapy integration via Spotify embed (WebView-style content)
- Community-sourced autism-friendly spaces
- History tracking and therapist-shareable CSV export
- Profile management for sensory preferences and caregiver contact
- AR-style navigation guidance simulation for desktop
